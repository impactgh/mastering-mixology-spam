# Spam Tech Strategy Guide

## What Makes This Strategy Special?

The spam tech method is brilliant because it **pre-processes potions at the "wrong" stations**. This means you have every recipe type ready at every station, giving you instant flexibility.

## The 27-Potion Setup

### Recipe Distribution
```
ALA (AquaLux Amalgam)     - 3 potions (1 per station)
ALL (Anti-Leech Lotion)   - 3 potions (1 per station)
AAM (Azure Aura Mix)      - 3 potions (1 per station)
MMA (Mystic Mana Amalgam) - 3 potions (1 per station)
MLL (MegaLite Liquid)     - 3 potions (1 per station)
MMM (Mammoth-Might Mix)   - 3 potions (1 per station)
  OR LLL (LipLack Liquor) - 3 potions (1 per station)
MML (Marley's MoonLight)  - 3 potions (1 per station)
MAL (MixALot)             - 6 potions (2 per station)
---
TOTAL: 27 potions
```

### Station Breakdown
Each of the 3 stations should have:
- 1x ALA
- 1x ALL
- 1x AAM
- 1x MMA
- 1x MLL
- 1x MMM (or LLL)
- 1x MML
- 2x MAL

## Why Process at "Wrong" Stations?

Normally you'd only process:
- **Alembic**: AAA, AAM, ALA, ALL
- **Retort**: LLL, MML, MLL
- **Agitator**: MMM, MMA, MAL

But with spam tech, you process EVERYTHING at EVERY station!

### Example: MMA (Mystic Mana Amalgam)
- "Correct" station: Agitator (Homogenise)
- Spam tech: Also make it at Alembic (Crystalise) and Retort (Concentrate)

This means when an order asks for MMA, you can submit it instantly regardless of which station you just used!

## Step-by-Step Execution

### Phase 1: Preparation (5-10 minutes)
1. Bank your herbs and make paste (Mox, Aga, Lye)
2. Fill the hopper with paste
3. Open the plugin panel

### Phase 2: Making Potions (10-15 minutes)
1. Follow the plugin's "Next" recommendation
2. Mix the potion at the mixing vessel
3. Process it at the recommended station
4. Click the button in the plugin panel
5. Repeat until you have 27 potions

### Phase 3: Spam Conveyor (30 seconds)
1. When plugin says "SPAM CONVEYOR!"
2. Stand next to the conveyor belt
3. Spam click it as fast as possible
4. Submit all 27 potions rapidly
5. Collect your massive XP and points!

### Phase 4: Repeat
1. Plugin resets automatically
2. Start making the next batch

## Tips & Tricks

### Efficient Paste Usage
- Use cheap herbs for paste:
  - Tarromin for Mox (13.8 gp/paste)
  - Lantadyme for Aga (46.9 gp/paste)
  - Avantoe for Lye (69.6 gp/paste)

### Station Pathing
- Group your potion-making by station to minimize running
- Suggested order: Alembic → Retort → Agitator → Conveyor

### Processing Speed Boosts
- **Agitator**: Spam click at the sound cue (66% faster)
- **Alembic**: Click on 5th pump (33% faster)
- **Retort**: Click every tick (50% faster)

### Digweed Usage
- Save Digweed for MAL potions only
- Doubles XP and points
- Use before processing, not after

## Common Mistakes

The plugin helps with this - use it!

## Advanced: Customization

You can adjust the recipe mix based on your goals:

### More MAL (Best Points)
- Reduce other recipes to 2 per station
- Increase MAL to 3 per station
- Total: 24 + 9 MAL = 33 potions (if inventory space allows)

### Faster Setup
- Skip one recipe entirely (e.g., skip MMM)
- Make 24 potions instead of 27
- Slightly less efficient but faster to prepare

### Point Targeting
- If you need more Mox: Add extra MMA, MML, MLL
- If you need more Aga: Add extra AAM, ALA, ALL
- If you need more Lye: Add extra MML, MLL, ALL
